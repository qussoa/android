   # 단위 문제
    
   * px(pixel)
    dpi(dot per inch) 1인치에 들어가는 점의수가 결정되어
    화면크기에 따라 이미지 크기나 선명도가 변화되는 단위
    
   * sp(scaled pixel) 
    크기에 독립적인 화소 주로 font 관련하여 사용하는 단위
    
   * dp(device indepent)
    화면의 물리적 밀도에 기반해서 생선하는 단위
    안드로이드에서 이미지 등을 표시할 떄 권장하는 단위
    dp 단위를 적용시 실제 보이는 이미지의 크기가 동일하게 보이는 효과
